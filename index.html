Ruby Web Projects

Repair Buddy

This app uses the Edmunds Developer API to supply data about cars, which allows users to save maintenance and repair records for their vehicles. This app is still being developed and at this point provides support for creating cars and records. The intention is to include repair records sourced from the Edmunds API and use the past records to generate service reminders.

Flight Pinner

This project provides an interface for users to sign up and "log" flights they've taken. It sources its airport, airline, and flight route data from http://openflights.org. This project demonstrates the usage of the Google Maps Javascript API and the GMaps4Rails gem. It also demonstrates the usage of the Geocoder gem for geocoding airports. All testing is implemented using RSpec and Capybara.

Source

Sinatra Chess

This project is a web-based share-the-keyboard-with-your-opponent game of chess. The chess library which powers the game is written in Ruby and contained in a Gem which I also authored. This project originated out of a coding challenge to build a command line game of chess. After completing this, I decided to get it up and running with a web-based UI powered by the Sinatra DSL.

NYC Housing Maps

Rails application using Google Maps and the GeoXML3.js library for displaying housing data in New York City. The application currently shows median rent by zipcode and allows selection of dates from which data is available. Data is supplied from Zillow using the Quandl API. To support the GeoXML3 library, I gemified the JavaScript asset.
